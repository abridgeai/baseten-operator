# Security Policy

## Status

This project is in early open-source development. We take security seriously but do not currently guarantee response times or patch timelines.

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public GitHub issue**.

Instead, open a [GitHub Security Advisory](https://github.com/abridgeai/baseten-operator/security/advisories/new) on this repository. This keeps the report private until a fix is available.

## Scope

This policy covers the baseten-operator codebase only. For vulnerabilities in the Baseten platform, contact Baseten directly at https://www.baseten.co/security.
